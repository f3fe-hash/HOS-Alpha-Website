EXIT(1)

NAME

exit — terminate the current shell or session

SYNOPSIS

exit [<status>]

DESCRIPTION

Ends the current terminal session or shell. Optionally, a numeric exit status can be provided.

ARGUMENTS

  • <status> — optional exit status code (default is 0)

EXAMPLES

# Exit with status 0 (success)
exit

# Exit with status 1 (error)
exit 1

OUTPUT

Closes the current terminal or shell session.

EXIT STATUS

  • 0 — success (default)
  • any other number — returned as exit status
