HELP(1)

NAME

help — display help information for HOS commands

SYNOPSIS

help [<command>]

DESCRIPTION

Shows usage information and details about the specified command. If no command is given, lists all available commands.

ARGUMENTS

  • <command> — the command to get help for (optional)

EXAMPLES

# Show general help listing
help

# Show help for the 'run' command
help run

OUTPUT

Displays help text in the terminal.

EXIT STATUS

  • 0 — success
  • 1 — unknown command specified
