Ping

Description: Pings a host a certain number of times with a timeout.

Arguments: <packets> <packet size> <hostname / IP> <timeout (microseconds, 10-6 sec)>

Usage: ping <packets> <packet size> <hostname / IP> <timeout>

Example: ping 10 64 8.8.8.8 10000

Output: Displays the ping results.
