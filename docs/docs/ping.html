PING(1)

NAME

ping — send ICMP echo requests to network hosts

SYNOPSIS

ping <host> [count]

DESCRIPTION

Sends ICMP echo request packets to a specified host to check connectivity and measure round-trip time. Optionally specify the number of requests to send.

ARGUMENTS

  • <host> — hostname or IP address to ping
  • [count] — optional number of ping requests (default is unlimited)

EXAMPLES

# Ping google.com indefinitely
ping google.com

# Ping 8.8.8.8 five times
ping 8.8.8.8 5

OUTPUT

Displays ICMP reply details including response times.

EXIT STATUS

  • 0 — success
  • 1 — host unreachable or error
