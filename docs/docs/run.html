RUN(1)

NAME

run — execute a package installed via HPM

SYNOPSIS

run <package-name>

DESCRIPTION

Executes the specified HOS package that was previously installed using the hpm command. This will run the package's main executable, assuming it's present and marked as runnable.

ARGUMENTS

  • <package-name> — The name of the package to execute.

EXAMPLES

# Run a package named 'dummy'
run dummy

OUTPUT

If successful, the package will begin execution in the terminal or GUI, depending on its type.

EXIT STATUS

  • 0 — success
  • 1 — failed to locate or execute package

SEE ALSO

  • hpm(1) — package installer
