SETPORT(1)

NAME

setport — configure the network port for HOS services

SYNOPSIS

setport <port-number>

DESCRIPTION

Sets the network port that HOS services will listen on. Useful for changing default ports or avoiding conflicts.

ARGUMENTS

  • <port-number> — the TCP or UDP port number to set (1-65535)

EXAMPLES

# Set port to 8080
setport 8080

OUTPUT

Confirms the port has been set or reports errors.

EXIT STATUS

  • 0 — success
  • 1 — invalid port number or error
