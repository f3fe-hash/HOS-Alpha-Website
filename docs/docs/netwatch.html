NETWATCH(1)

NAME

netwatch — monitor network packets on an interface

SYNOPSIS

netwatch <interface> <max hosts>

DESCRIPTION

Monitors all network packets passing through the specified network interface. Displays details about hosts and traffic, limited to the maximum number of hosts specified.

ARGUMENTS

  • <interface> — the network interface to monitor (e.g., wlo0, eth0)
  • <max hosts> — maximum number of hosts to display

EXAMPLES

# Monitor wireless interface wlo0, showing up to 15 hosts
netwatch wlo0 15

OUTPUT

Continuously displays packet information and host activity on the selected interface. Output is updated in real time, showing packet counts, host IPs, and other network data.

EXIT STATUS

  • 0 — monitoring started successfully
  • 1 — invalid interface, permission error, or other failure
