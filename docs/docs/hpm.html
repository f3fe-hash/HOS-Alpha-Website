HPM(1)

NAME

hpm — HOS package manager for installing, removing, and managing packages

SYNOPSIS

hpm <command> [args, usually package]

DESCRIPTION

Manages HOS packages including installation, removal, and updates. Supports commands like install, purge, download, and update.

ARGUMENTS

  • <command> — subcommand to run (install, remove, list, update)
  • <args> — arguments for the subcommand

EXAMPLES

# Download a package named 'dummy' (This is a real package)
hpm download dummy

# Install a package named 'dummy'
hpm install dummy

# Update a package
hpm update dummy

# Purge a package
hpm purge dummy

OUTPUT

Varies depending on subcommand, typically status messages and package lists.

EXIT STATUS

  • 0 — success
  • 1 — failure due to invalid command or package
