HPM(1)

NAME

hpm — HOS package manager for installing, removing, and managing packages

SYNOPSIS

hpm <command> [<args>]

DESCRIPTION

Manages HOS packages including installation, removal, and updates. Supports commands like install, remove, list, and update.

ARGUMENTS

  • <command> — subcommand to run (install, remove, list, update)
  • <args> — arguments for the subcommand

EXAMPLES

# Install a package named 'examplepkg'
hpm install examplepkg

# Remove a package
hpm remove examplepkg

# List installed packages
hpm list

OUTPUT

Varies depending on subcommand, typically status messages and package lists.

EXIT STATUS

  • 0 — success
  • 1 — failure due to invalid command or package
