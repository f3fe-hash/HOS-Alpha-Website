HOS Alpha

A lightweight, experimental Operating System built from scratch

What is HOS Alpha?

HOS Alpha is a hobby operating system focused on simplicity, performance, and learning. Built from the ground up in low-level languages, it's designed to explore core OS concepts and push the limits of bare-metal programming.

  • Boots on real hardware and emulators (QEMU/Bochs)
  • Basic kernel and interrupt handling
  • Custom memory management (WIP)
  • Written in C/C++ with minimal dependencies

üõ†Ô∏è Getting Started

To build and run HOS Alpha, follow these steps:

  1. Download the OS (Section Above). Either Pre-compiled (Go to #5), or from source (Preferred method)
  2. Install dependancies:

    1. Linux:

      Run: 
      
sudo apt update && sudo apt upgrade -y && sudo apt install cmake make libssl-dev

    2. MacOS:

      Run: 
      
brew update && brew upgrade && brew install cmake make openssl

    3. Windows:

      1. Download the source code from cmake.org/download
      2. Extract it and open a terminal (PowerShell, Git Bash, or MSYS2)
      3. Navigate to the extracted folder and run: 
        
cd path/to/cmake-source
bootstrap
make
make install
      4. Alternatively, open the .sln file in Visual Studio and build it from there
      5. Verify installation: 
        
cmake --version
  3. Build the OS:
    1. Open a terminal and navigate to the HOS Alpha directory
    2. Run the following command: 
      
source compile.sh
  4. Run the OS: 
    
source compile.sh run
  5. Pre-compiled: Go to the Downloads page and download the pre-compiled version of HOS Alpha. Extract the zip file to get the raw compiled files.

    To run the OS type: 

    
cd /path/to/HOS-Alpha
./HOS
