HOS Alpha

A lightweight, experimental Operating System built from scratch

What is HOS Alpha?

HOS Alpha is a hobby operating system focused on simplicity, performance, and learning. Built from the ground up in low-level languages, it's designed to explore core OS concepts and push the limits of bare-metal programming.
